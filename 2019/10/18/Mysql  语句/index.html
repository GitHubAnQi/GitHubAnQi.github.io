Mysql 语句

添加数据：

insert [into] 表名 （列名称1，列名称2，列名称3） values (添加的值);

修改数据：

upade 表名 set 需要修改的字段 /while 条件；

删除数据：

delete from 表名 /while 条件；

truncate 表名 ；

查询数据

select * from 表名；全表查询

select 列名 from 表名；查询列

select 字段名1 as 别名，字段名2 as别名 from 表名; 查询列 更改列的字段名为别名

Mysql内置函数

concat 连接字符串 ；

concat 连接字符串 select concat(‘今天’，‘晴天’’)，返回’今天晴天‘；

length 字符串长度； length(‘hello world’);返回结果11

upper 字符串转为大写；upper(‘abcd’)返回大写ABCD

ltrim 去除字符串左边的空格；

rtrim 去除字符串右边的空格；

trim 去除字符串左右两边的空格；

replace replace（’abccd,’c’,’x’）返回为’abxxd‘替换字符串中的c 换成x；

strcmp 比较字符；

substring
